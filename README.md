## 🌟 Présentation

**Orlyne** est une IA de nouvelle génération créée par **Syntaxe Tech**. 
Elle combine la puissance de **Llama 3 8B** avec une personnalité chaleureuse et des capacités de codage multi-langage exceptionnelles.

### 🎯 Caractéristiques uniques

- **🧠 Personnalité amicale** - Comme ChatGPT, mais en mieux !
- **💻 Code multi-langage** - Python, JavaScript, Java, C++, Rust, Go, PHP, Ruby, SQL, Bash...
- **🔧 Débogage intelligent** - Trouve et corrige les erreurs automatiquement
- **📚 Apprentissage continu** - S'améliore avec vos feedbacks
- **🌐 Interface web/mobile** - Accessible partout
- **⚡ Optimisé pour 8GB RAM** - Version quantifiée 4-bit
